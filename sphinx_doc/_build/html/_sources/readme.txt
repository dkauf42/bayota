If you want to use this package, then run main.py

This can be run either with or without a GUI.

-- To run without a GUI -- 
Comment out the code in OptSandbox.util.Runner
that generates the tkinter root window.
Then, scenario options must be specified by a comma separated value table.

-- To run with a GUI -- 
Uncomment the code in OptSandbox.util.Runner
that generated the tkinter root window.
